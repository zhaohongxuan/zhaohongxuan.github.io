二、子组件向父组件传值

子组件向父组件与父组件给子组件传值类似，假如组件Right要传值给Root,
Root将传递一个函数greeting给子组件Right，子组件Right调用该函数，将想要传递的信息，作为参数，传递到父组件的作用域中。
函数将保障子组件Right在调用 greeting函数时，其内部 this 仍指向父组件。
